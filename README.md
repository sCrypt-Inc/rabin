# A Javascript Library for Rabin Signatures
[Rabin signature algorithm](https://en.wikipedia.org/wiki/Rabin_signature_algorithm) is a method of digital signature.

## Installation

```bash
npm install rabinsig
```

## Run example

```bash
npm run example
```
